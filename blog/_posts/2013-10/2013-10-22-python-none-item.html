python去除列表中的空字符串

今天写代码时遇到需要去除python列表中的空字符串的操作，总结了以下几种方法

filter(lambda)

list = ['','12','34','']
list = filter(lambda x:x != '', list)

for循环

list = ['','12','34','']
list = [ i for i in list if i != '']

while循环

list = ['','12','34','']
while '' in list:
    list.remove('')
